# Quantum Hydra Solutions Website

Welcome to the official GitHub repository for the **Quantum Hydra Solutions** company website. This is a fully responsive, professional, multi-page website built with **HTML, CSS, JavaScript, and Bootstrap 5**. It is designed for **GitHub Pages hosting**, allowing easy deployment for static web hosting.

---

## **Website Overview**

**Quantum Hydra Solutions** is a pioneering technology company specializing in:

- Web Development & Design  
- Software & ICT Solutions  
- IoT & Smart Systems  
- Cybersecurity & Data Protection  
- Futuristic Technology Solutions (AI, Robotics, Experimental Tech)

**Key Features of the Website:**

- Fully responsive design (mobile, tablet, desktop)  
- Multi-page structure:  
  - **Home** – Hero section, company expertise  
  - **About** – Introduction, vision, mission, history  
  - **Services** – Detailed services overview  
  - **Products** – Product cards with images and descriptions  
  - **Contact** – Contact form integrated via Formspree  
- Modern cards, hover effects, and professional layout  
- Bootstrap 5 framework for consistency and responsiveness  
- Internal navigation with a sticky navbar  
- Footer with social media links and copyright
